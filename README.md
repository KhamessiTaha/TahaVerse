## ğŸš€ Quick Start

### Prerequisites
```bash
Node.js >= 18.0.0
npm or yarn
```

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/khamessitaha/TahaVerse.git
   cd TahaVerse
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Production Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
